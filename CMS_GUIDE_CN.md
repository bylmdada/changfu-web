# Google Sheets 內容管理系統 (CMS) 操作手冊

本系統設計讓您可以直接透過 **Google Sheets (試算表)** 來管理網站上的「最新消息」、「人才招募」與「課程資訊」，無需撰寫任何程式碼。

---

## 🚀 1. 系統初始化設定 (僅需執行一次)

### 步驟一：建立您的資料庫
1. 在您的 Google Drive 建立一個新的 Google Sheet。
2. 將專案中的 `cms_templates` 資料夾內的 CSV 檔案內容，分別匯入到三個不同的工作表 (Tabs)：
   - **分頁名稱**：`news` (對應最新消息)
   - **分頁名稱**：`jobs` (對應人才招募)
   - **分頁名稱**：`courses` (對應課程資訊)

### 步驟二：發布資料
為了讓網站讀取資料，您需要將試算表發布到網路：
1. 在 Google Sheets 選單點選 **檔案 (File)** > **共用 (Share)** > **發布到網路 (Publish to web)**。
2. 針對每個分頁 (news, jobs, courses) 分別操作：
   - 選擇分頁名稱 (例如: "news")。
   - 格式選擇 **逗號分隔值 (.csv)**。
   - 點擊 **發布 (Publish)**。
   - **複製產生的連結**。

### 步驟三：連結至網站
1. 開啟專案中的 `js/config.js` 檔案。
2. 將剛剛複製的 CSV 連結，分別貼入對應的欄位中：
   ```javascript
   window.CONFIG = {
       sheetUrls: {
           news: '您的_NEWS_CSV_連結',
           jobs: '您的_JOBS_CSV_連結',
           courses: '您的_COURSES_CSV_連結'
       },
       // ...
   };
   ```

## 📝 2. 如何新增/修改內容

### 📢 最新消息 (News)
在 `news` 分頁中，每一列代表一則新聞。

| 欄位名稱 | 說明 | 範例格式 |
| :--- | :--- | :--- |
| **id** | 唯一識別碼，不可重複 | `news-20240201` |
| **title** | 新聞標題 | `春節服務時間公告` |
| **category** | 分類 (公告/活動/課程/徵才) | `公告` |
| **date** | 發布日期 | `2026-02-10` |
| **summary** | 列表頁顯示的短摘要 | `因應春節假期...` |
| **content** | 內文 (支援 HTML 標籤) | `<p>內文段落...</p>` |
| **coverImage** | 封面圖片網址 (可留空) | `https://example.com/img.jpg` 或 Emoji `📢` |
| **status** | 狀態 | `published` (發布) / `draft` (草稿) |

### 💼 人才招募 (Jobs)
在 `jobs` 分頁中，每一列代表一個職缺。

| 欄位名稱 | 說明 | 範例格式 |
| :--- | :--- | :--- |
| **id** | 唯一識別碼 | `job-001` |
| **title** | 職位名稱 | `照顧服務員` |
| **location** | 工作地點 | `冬山日間照顧中心` |
| **type** | 職務類型 | `全職` / `兼職` |
| **status** | 狀態 | `active` (徵才中) / `closed` (已關閉) |
| **salary** | 薪資範圍 | `32k-38k` |
| **requirements** | 徵求條件 (換行請用 `\n`) | `證照\n經驗` |
| **duties** | 工作內容 (換行請用 `\n`) | `照顧\n清潔` |
| **benefits** | 員工福利 (換行請用 `\n`) | `勞健保\n三節` |

### 🎓 課程資訊 (Courses)
在 `courses` 分頁中，每一列代表一堂課程。

| 欄位名稱 | 說明 | 範例格式 |
| :--- | :--- | :--- |
| **id** | 唯一識別碼 | `course-001` |
| **title** | 課程名稱 | `照顧服務員訓練` |
| **category** | 課程類別 | `專業課程` |
| **status** | 狀態 | `open` (報名中) / `full` (額滿) / `closed` (截止) |
| **startDate** | 開始日期 | `2026-03-01` |
| **endDate** | 結束日期 | `2026-04-15` |
| **hours** | 時數 | `90` |
| **location** | 上課地點 | `冬山教室` |
| **fee** | 費用 | `免費` |
| **quota** | 名額 | `30` |
| **description** | 課程說明 | `課程詳細介紹...` |

## 📋 3. 表單設定 (Google Forms)

網站上的「聯絡我們」、「求職申請」、「課程報名」按鈕會連結到 Google 表單。

1. 建立對應的 Google 表單。
2. 點擊 **傳送 (Send)** > **連結圖示** > **縮短網址** > **複製**。
3. 將連結更新至 `js/config.js`：
   ```javascript
   forms: {
       contact: '您的_聯絡表單_連結',
       jobApply: '您的_求職表單_連結',
       courseRegister: '您的_報名表單_連結'
   }
   ```

## ❓ 常見問題 (FAQ)

**Q: 修改了 Google Sheet，但網站沒有更新？**
A: Google Sheets 的發布有快取機制，通常需要等待 **3~5 分鐘** 才會反映到網站上。您可以嘗試重新整理網頁或清除瀏覽器快取。

**Q: 圖片無法顯示？**
A: 請確保 `coverImage` 欄位填入的是圖片的 **直接連結 (Direct Link)**，結尾通常是 `.jpg` 或 `.png`。如果是 Google Drive 的圖片，需轉換為直接連結格式。

**Q: 如何暫時下架某個內容？**
A: 將該列的 `status` 欄位改為其他值（例如 `draft` 或 `closed`），網站就會自動過濾掉該筆資料。
